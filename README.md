<h2> 🩵오늘의집🩵의 핵심 기능 </h2>

<h3> 1️⃣ Home - 홈 </h3>
<img width="650" alt="Untitled-8" src="https://github.com/GO-SOPT-GROUP5/ohouse-client/assets/81505421/5cfd007e-d6f1-4567-bfef-9420dc1b7808">

- Gnb, Lnb의 네비게이션 헤더로 페이지 이동이 가능합니다
- 메인 화면의 메뉴 배너들을 통해 해당 **메뉴 페이지로 이동**이 가능합니다
- 하단의 **광고 배너**는 자동으로 슬라이드됩니다

<h3> 2️⃣ List - 등록 매물 확인하기 </h3>
<img width="650" alt="Untitled-9" src="https://github.com/GO-SOPT-GROUP5/ohouse-client/assets/81505421/eb1878e2-5e64-4502-bc0f-4bfa53746f93">

- 등록된 매물 목록을 조회할 수 있습니다
  - 매물 목록의 데이터는 **무한 스크롤**이 가능하도록 불러와집니다.
  - 매물은 전체/월세/전세/매매 **카테고리별로 필터링**이 가능합니다.
  - 매물은 최신순/별점순/좋아요순으로 **정렬**이 가능합니다.
- `+` 박스 클릭 시, 매물 등록 페이지로 이동합니다. 
- `체크리스트 내역보기` 버튼 클릭 시, 체크리스트 조회 페이지로 이동합니다.

<p>
<img width="400" alt="스크린샷 2023-05-26 오후 6 52 10" src="https://github.com/GO-SOPT-GROUP5/ohouse-client/assets/81505421/36660dff-a297-4df1-acf8-ef3063f1678a">
<img width="400" alt="스크린샷 2023-05-26 오후 6 52 37" src="https://github.com/GO-SOPT-GROUP5/ohouse-client/assets/81505421/f75f7cbb-6a66-46b6-ad70-c496794dff75">
</p>

- `...` 버튼 클릭 시, **더보기 모달**이 나타납니다.
  - **매물 수정** : `수정하기` 클릭 시 해당 매물의 체크리스트 수정 페이지로 이동
  - **매물 삭제** : `삭제하기` 클릭 시 `삭제 재확인 모달`이 나타나며, `확인` 클릭 시 매물이 삭제됩니다.
 

<h3> 3️⃣ Help - 서비스 설명 </h3>
<img width="650" alt="Untitled-10" src="https://github.com/GO-SOPT-GROUP5/ohouse-client/assets/81505421/81eb7bc8-b66e-40a1-b8ef-65489efabac0">

- 5개의 서비스 설명 카드가 `>` 버튼 클릭 시 **슬라이드**되어 보여집니다.

<h3> 4️⃣ Checklist - 집보기 체크리스트 </h3>
<img width="650" alt="스크린샷 2023-05-26 오후 6 38 23" src="https://github.com/GO-SOPT-GROUP5/ohouse-client/assets/81505421/078fdad0-24d3-43fd-a738-517310b0a5a0">
<img width="650" alt="Untitled-14" src="https://github.com/GO-SOPT-GROUP5/ohouse-client/assets/81505421/a25bc141-9a91-473f-a7c6-ca228b4d0479">

- 매물을 등록하기 위해 체크리스트 정보를 입력합니다.
- 최초의 매물제목은 **등록 시간으로 자동 설정**되며, `✏️`버튼 클릭 시 상단의 항목들을 **수정**할 수 있습니다.
- 계약형태(전세/월세/매매) 선택에 따라 추가 정보 입력 뷰가 하단에 다르게 추가됩니다. 
- 체크리스트의 카테고리는 **드롭다운**으로 구성되어, 클릭 시 세부 카테고리를 선택할 수 있습니다.
